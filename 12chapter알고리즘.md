# 알고리즘 기초

## 대표적인 알고리즘 종류

| 종류 | 설명 |
|---|---|
| 정렬 | 컨테이너 원소 정렬 |
| 검색 | 원소 존재 확인, 특정 원소 검색 |
| 변환 | 원소 수정, 다른 형식으로 변환 |
| 반복자 | 컨테이너 원소 순회 및 조작 |
| 집계 | 컨테이너 원소 집계 |

---

## 시간 복잡도 표기법

### 3가지 유형

| 표기법 | 의미 |
|---|---|
| 빅-오메가 $\Omega$ | 최선(Best case) 연산 횟수 |
| 빅-세타 $\Theta$ | 평균(Average case) 연산 횟수 |
| 빅-오 $O$ | 최악(Worst case) 연산 횟수 |

> 코딩 테스트에서는 **빅-오 O(n)** 기준으로 수행 시간 계산
> 모든 테스트 케이스를 통과해야 합격이므로, **최악의 경우를 기준**으로 판단

### 계산 규칙
1. 상수는 시간복잡도 계산에서 제외
2. 가장 많이 중첩된 반복문이 기준

## 정렬 알고리즘

| 알고리즘 | 시간복잡도 | 설명 |
|---|---|---|
| 버블 (Bubble) | $O(n^2)$ | 인접 데이터 비교·swap 반복 |
| 선택 (Selection) | $O(n^2)$ | 최솟값 선택 후 정렬 위치에 배치 |
| 삽입 (Insertion) | $O(n^2)$ | 정렬된 부분에 원소 삽입 |
| 퀵 (Quick) | $O(n \log n)$ | 피벗 기준 분할 정복 |
| 병합 (Merge) | $O(n \log n)$ | 분할 후 병합 |
