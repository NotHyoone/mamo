# Java 수업 정리 (2025.04.20 — IT Academy)

## 클래스 기본 개념

> 구조체 + 메서드(기능) = **클래스**

---

## 접근 제어자

| 접근 제어자 | 접근 범위 |
|---|---|
| `public` | 모든 클래스 |
| `private` | 현재 클래스 내부만 |
| `protected` | 동일 패키지 + 상속 관계 하위(자식) 클래스 |
| `default` (package-private) | 동일 패키지 내 클래스만 (C++에는 없음) |

---

## Java 메모리 구조

- **스택 (Stack)** — 메서드 호출, 지역 변수
- **힙 (Heap)** — 객체 (new 로 생성된 인스턴스)
- **클래스 (Method Area)** — 클래스 정보, static 변수

---

## 생성자

- **디폴트 생성자** — 매개변수 없음, 객체 기본 초기화
- **매개변수 생성자** — 인수를 받아 객체 초기화

---

## 메서드 용어

- **인수 (Argument)** — 함수 호출 시 전달하는 값
- **매개변수 (Parameter)** — 함수가 인수를 받아주는 변수
- **일반 메서드** — 기능을 담는 메서드
